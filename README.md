
# Notes

Personal knowledge base with persistent wiki and archived documents.

## Structure

- **`llm-wiki/`** — Karpathy-style persistent Markdown wiki. Source materials → cross-linked wiki pages.
- **`history/`** — Archived historical documents (e.g., `AI/`).
- **`docs/`** — Project documentation.
- **`skills/`** — Claude Code skill definitions.

## llm-wiki

See [`llm-wiki/README.md`](llm-wiki/README.md) for details on the wiki system, tools, and workflows.
