## Definition

Eric Evans 是《Domain-Driven Design》一书的作者，Domain-Driven Design 方法论的创始人，专注于帮助团队构建复杂的软件系统。

## Key Attributes

| 属性 | 值 |
|------|-----|
| 领域 | 软件架构、业务建模 |
| 核心贡献 | DDD 方法论 |
| 著作 | Domain-Driven Design (2003) |

## Core Contributions

- **战略设计** — Bounded Context、Subdomain、Context Map
- **战术设计** — Entity、Value Object、Aggregate、Repository、Domain Event
- **Ubiquitous Language** — 业务与技术团队共享的统一语言
