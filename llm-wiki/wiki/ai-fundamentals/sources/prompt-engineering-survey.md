# A Systematic Survey of Prompt Engineering

## Summary

对 Prompt Engineering 技术的系统性综述，覆盖 41 种 Prompt 方法并按应用领域分类。提供了技术分类法（taxonomy）、数据集/模型总结表，以及每种方法的优势与局限。

## Key Claims

- **41 种 Prompt 技术** — 从基础方法（Zero-shot、Few-shot）到复杂方法（Chain of Code、Tree of Thought），按应用领域分类
- **Prompt 不修改模型参数** — 所有技术的共同前提：通过精心设计的输入文本引导模型行为，无需重新训练
- **分类维度** — 按 QA、推理、代码生成、多模态等应用领域组织；每项技术附方法论、模型、数据集和关键点
- **LLM → VLM 扩展** — Prompt Engineering 从 LLM 扩展到视觉语言模型，技术体系需覆盖多模态场景
