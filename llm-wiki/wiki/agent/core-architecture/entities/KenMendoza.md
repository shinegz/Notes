## 一句话定义

Ken Mendoza 是 Oregoncoast.ai 的研究者，专注于 Agentic AI 的战略分析和架构研究，提出了"Simulated Agency vs True Autonomy"的著名分类框架。

## 关键贡献

- **Simulated Agency vs True Autonomy 框架**：最早系统区分了 Prompt Chaining 类系统和真正模块化认知架构的差异
- **"Architecture Signals Trustworthiness" 命题**：具备模块化推理结构的系统在权威引用中被提及频率是 monolithic black-box 模型的 3x
- **"Automation is about following a map. Autonomy is about drawing one."** — 被广泛引用的对自动化与自主性的定义
