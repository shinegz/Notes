# 人工智能的发展历程

> 从图灵测试到 ChatGPT，AI 走过了七十余年。

## 30 秒心智模型

**核心问题：机器能否思考？**

1950 年图灵提出这个问题时，答案只能靠想象。七十多年后，答案已经写在我们每天使用的对话助手、翻译工具、代码补全里。

**关键转折点：**

| 时期 | 核心技术 | 代表成果 |
|------|----------|----------|
| 1950s-1960s | 符号逻辑 | 达特茅斯会议、ELIZA |
| 1970s-1980s | 专家系统 | MYCIN、XCON |
| 1990s-2010s | 统计学习 | SVM、深度学习 |
| 2017+ | Transformer | GPT、BERT、ChatGPT |

## 1. 起源：图灵的追问

1950 年，艾伦·图灵发表了《计算机器与智能》。这篇论文没有给出"什么是智能"的定义，而是抛出了一个可操作的测试：

> 如果一个人无法区分对话对象是人还是机器，就认为机器通过了测试。

这个被称为"图灵测试"的思想实验，成为后世 AI 研究的北极星。

六年后，1956 年的达特茅斯会议聚集了约翰·麦卡锡、马文·明斯基、克劳德·香农等人物。"人工智能"这个词正式诞生。

```
达特茅斯会议 (1956)
├── John McCarthy  → 首次提出 "Artificial Intelligence"
├── Marvin Minsky  → 神经网络先驱
├── Claude Shannon → 信息论之父
└── Allen Newell   → 物理符号系统假设
```

会议的目标很宏大：两个月内，让机器达到人类智能的水平。现实比理想骨感得多。

## 2. 第一次浪潮：符号主义的兴衰

符号主义（Symbolic AI）的思路很清晰：

> 智能 = 操作符号的规则系统

人类用逻辑推理解决问题，那机器就按照逻辑规则来。

### 早期成果

- **通用问题求解器（GPS）**：Newell 和 Simon 开发，能证明数学定理
- **ELIZA**：MIT 的聊天程序，模拟心理治疗师。技术简陋，但人们真的和它对话
- **SHRDLU**：能理解积木世界的自然语言指令

### 走向低谷

1973 年，英国政府发布《莱特希尔报告》，直接泼冷水：

- AI 承诺的功能大多无法实现
- 神经网络研究的期望被证伪
- 感知机连 XOR 这种基本逻辑都学不会

这一棒子下去，美国和英国都削减了 AI 经费。第一次 AI 寒冬降临。

## 3. 第二次浪潮：专家系统的局限

1980 年代，专家系统带来了短暂的复苏。

### 什么是专家系统

```
专家系统架构

┌─────────────────────────────────────────┐
│            用户界面                       │
└──────────────────┬──────────────────────┘
                   │
┌──────────────────▼──────────────────────┐
│            推理引擎                       │
│    (正向链 / 反向链 / 混合推理)           │
└──────────────────┬──────────────────────┘
                   │
      ┌────────────┴────────────┐
      ▼                         ▼
┌───────────┐            ┌───────────┐
│  知识库    │            │  规则库    │
│ (事实)     │            │ (IF-THEN) │
└───────────┘            └───────────┘
```

### 代表作

- **MYCIN**：诊断细菌感染的医疗专家系统，准确率接近人类专家
- **XCON**：DEC 公司的专家系统，负责配置计算机订单，节省了大量成本

### 寒冬再次降临

专家系统的致命缺陷：

1. **知识获取瓶颈**：把专家知识一条条写成规则，费时费力
2. **脆弱性**：遇到规则没覆盖的情况就崩溃
3. **维护成本**：规则越来越多，冲突和矛盾难以排查

1987 年， Lisp 机器市场崩塌，第二次 AI 寒冬开始。

## 4. 第三次浪潮：深度学习的突破

### 反向传播的复兴

1986 年，Rumelhart、Hinton 和 Williams 重新发现了反向传播算法。但真正的突破要等到 1990 年代末和 2000 年代——算力终于跟上了算法。

### ImageNet 时刻

2009 年，李飞飞团队发布了 ImageNet。2012 年，AlexNet 在 ImageNet 挑战赛上一举成名：

```
AlexNet vs 传统方法 (ILSVRC 错误率)

年份    传统方法    AlexNet
2010   28%         -
2011   26%         -
2012   -           16.4%  ← 深度学习的爆发
2013   -           11.7%
2014   -           6.7%
2015   -           3.6%   ← 超越人类水平 (~5%)
```

这不只是比胜负——它宣告了深度学习在大规模视觉任务上的统治地位。

### 自然语言处理的 CNN/RNN 时代

2013 年，Word2Vec 把词表示从 one-hot 推进到了稠密向量。2014-2015 年，Seq2Seq 模型和 Attention 机制让机器翻译突飞猛进。

LSTM 和 GRU 成为处理序列数据的主力。但它们有一个共同缺陷：

> **无法并行处理，必须一个词一个词地顺序计算。**

这个缺陷在 Transformer 出现后被彻底解决。

## 5. 第四次浪潮：Transformer 统治时代

2017 年，Google 发表了《Attention Is All You Need》。这篇论文的标题就是结论：

> 抛弃循环网络，注意力机制就够了。

### 注意力机制的核心思想

传统 RNN 要"记住"长序列很难——信息传得越远，衰减越严重。注意力机制让模型可以"随时回头看"任意位置的上下文。

```
注意力计算

Query (我想查询的信息)
    │
    ▼
┌─────────────────────────────┐
│   计算 Q 与所有 K 的相似度    │
└─────────────────────────────┘
    │
    ▼
┌─────────────────────────────┐
│   Softmax 归一化 → 注意力权重 │
└─────────────────────────────┘
    │
    ▼
┌─────────────────────────────┐
│   权重加权求和 V → 输出向量   │
└─────────────────────────────┘

Q、K、V 来自同一个输入的不同线性变换
```

### BERT 和 GPT 的分叉

2018 年，Google 和 OpenAI 分别发布了 BERT 和 GPT：

| | BERT | GPT |
|---|---|---|
| 架构 | 双向编码器 | 单向解码器 |
| 训练目标 | 掩码语言模型 (MLM) | 下一个词预测 |
| 擅长 | 理解任务 | 生成任务 |
| 代表 | 文本分类、问答 | 对话、写作、代码 |

2020 年 GPT-3 震惊业界：1750 亿参数，zero-shot 和 few-shot 能力令人咋舌——不需要微调，直接Prompt就能做任务。

### ChatGPT 时刻

2022 年 11 月，ChatGPT 上线。五天内用户破百万。

关键不是模型有多大，而是**对话界面**让普通人也能用 AI。

2023 年，GPT-4、Claude 2、Gemini、LLaMA 2 相继登场。开源和闭源模型同时狂奔。

## 6. 从 GPT 到 AGI 的想象

### 能力边界

大模型展示出了一些令人意外的能力：

- **涌现能力 (Emergent Abilities)**：模型大到某个规模后，突然具备了小模型没有的能力
- **思维链 (Chain of Thought)**：让模型"一步一步想"，推理能力显著提升
- **多模态**：能看图、会听话、能说图

### 未解决的难题

1. **幻觉**：一本正经地胡说八道
2. **实时知识**：训练数据截止后的事一概不知
3. **推理成本**：大模型的推理耗电惊人
4. **对齐问题**：让 AI 的目标和人类真正一致

### AGI 的迷雾

"通用人工智能（AGI）"还没有公认的定义。有人说 GPT-4 已经是 AGI，有人说离 AGI 还差得远。

真正的问题是：**我们不知道智能的本质是什么，所以也无法判断机器是否拥有了它。**

也许图灵在 1950 年就已经预言了这一切——与其争论定义，不如设计测试。

## 总结

```
人工智能发展脉络

1950 ─── 图灵测试 ─── 达特茅斯 ─── 符号主义 ─── 专家系统
  │                                                     │
  │                  ┌─────────────────────────────────┘
  │                  │
  └──────────► 深度学习 ────► Transformer ───► 大模型时代
```

三次浪潮教会我们：**过度承诺是 AI 最大的敌人，务实迭代才是正道。**

第四次浪潮的不同之处在于：大模型已经渗透进了日常工具。写代码有 Copilot，查资料有 ChatGPT，做设计有 Midjourney。这不再是实验室里的玩具——它是基础设施。
