# Agent 开发 学习资料汇总

**概念简述**：Agent（智能体 / LLM Agent）是基于大语言模型的 AI 系统，能根据自然语言输入独立完成多步任务；通常具备工具调用、上下文维持、规划与反思能力。可理解为：**Agent = 大模型 + 记忆 + 工具 + 规划 + 行动**。与传统单轮聊天不同，Agent 能在多步中维持状态、调用外部 API、并根据反馈调整行为。

## 速查与按人群推荐

- **零基础 / 先建立概念**：先看「概念简述」+ IBM/AI21 词条 + 《Building AI Agents In 44 Minutes》视频；中文可看「Python 轻松实现你的第一个 AI Agent」。
- **动手做第一个 Agent**：OpenAI Agents SDK 文档 + 《Build an AI Agent From Scratch in Python》或《Build Your First AI Agent With Google ADK》；中文可跟「从 0 到 1 搭建 Agent」。
- **系统学习（含责任与工程）**：Coursera「AI Agent Developer Specialization」或「AI Agents and Agentic AI in Python」。
- **多 Agent / 框架选型**：LangGraph 文档 + 「Comparing LangChain vs CrewAI vs AutoGen」；要接 MCP 时看 MCP 官方教程 + OpenAI Agents SDK 的 MCP 章节。
- **深入 ReAct 与循环设计**：「ReAct：最经典的 Agent 设计框架」+ LangGraph 的 ReAct 教程。
